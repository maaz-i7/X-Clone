X (Twitter) UI Clone — Pure Tailwind CSS

A frontend UI clone of X.com built entirely within a single `index.html` file. This is a hardcore styling experiment pushing Tailwind CSS to its absolute limits 
using 100% raw HTML and utility classes—zero JavaScript frameworks and zero custom CSS.
It features a fully responsive layout, with multiple break points for adjusting responsiveness in all devices, all handled via Tailwind's JIT compiler.

Check it out here - https://x-clone-by-maaz.vercel.app/
